Using SimplePluginÂ¶

To use SimplePlugin in your plugin, first you need to install SimplePlugin module addon in Kodi. An installable ZIP can be downloaded from Releases tab of this repository. I also plan to submit SimplePlugin module addon to the Kodi official repo.

Then you need to add the module in the requires section of your plugin’s addon.xml file as a dependency:

<requires>
  ...
  <import addon="script.module.simpleplugin" version="1.6"/>
  ...
</requires>

After that you can import SimplePlugin classes in your plugin’s Python code and use them as necessary.
